Key Design Decisions

  • TouchDetector is the only component touching private API — swap-friendly
  • GestureRecognizer is pure logic — no system dependencies, fully testable
  • ActionExecutor uses only public CGEvent API — stable
  • ConfigStore is shared between the settings UI and the runtime pipeline
