Design approach

  • Search bar: hidden by default, auto-appears when gesture count ≥ 6 (threshold configurable)
  • Scroll: gesture list is always in a ScrollView — invisible when few, natural when many
  • Same component: both states use the same SwiftUI List — no conditional layouts
  • For v1: we ship the "Now" version — the scroll and search are just SwiftUI List properties we turn on later
